# PersonalMediaSync

**An Android app that backs up photos and videos to a remote server.**

---

## Features
- Automatically scans device for images and videos using **MediaStore**
- Uploads media files to a **FastAPI backend**
- Supports **images and videos**
- Uploads run in **background threads** to keep the UI responsive
- Shows **upload status** with Toast messages
